API для новостного портала

Инструкции по запуску:

1. скачайте репозиторий
2. перейдите в файл \NewsPortal\settings.py
3. найдите блок  # Email settings
4. укажите свою mail.ru почту, с нее будут отправляться сообщения
5. откройте в консоли папку проекта и введите docker-compose up
6. перейдите по ссылке localhost:8080/swagger
